HTML

HTML (Hypertext Markup Language) es un lenguaje de marcado que permite definir la estructura y contenido de un documento usando marcas o etiquetas. HTML nos permite crear documentos interconectados. Un documento HTML es un archivo de texto plano escrito usando HTML

CSS

CSS es un lenguaje de hojas de estilo que permite modificar la apariencia, de manera selectiva, a los elementos de un documento HTML.
