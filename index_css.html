HOJAS DE ESTILO
Estilo Descripcíon
ESTILOS EN EL ELEMENTO Los estilos se incorporan en la etiqueta de inicio de un elemento usando el atributo style.
INCRUSTADOS EN EL DOCUMENTO
  • El elemento style contiene los estilos de un documento.
  • El elemento se incluye dentro del elemento head.
  • Incluir elementos usando el elemento style no es la forma mas eficiente de trabajar (no permite reutilizar el código ).
ESTILOS EN UNA HOJA DE ESTILOS EXTERNA
  • Una hoja de estilos externa, no es más que un archivo de texto que contiene todas nuestras reglas.
  • Es un archivo que tiene una extensión css.
  • Podemos darle cualquier nombre al archivo, cuidando que este sea descriptivo (formulario.css, global.css, etc.)
  • Luego de crear una hoja de estilos, necesitamos conectarla con la página que queremos dar formato. Para esto usamos el elemento link
  • El atributo href especifica la URL del recurso enlazado.
  • El atributo rel indica la relación del documento enlazado con el actual.
COLORES
Formato Descripcíon Regla Documentacíon
Nombres

Podemos especificar un color usando un nombre predefinido.

Modelo RGB
  • Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios.
  • Podemos definir la transparencia de un color usando la función rgba, esta función es idéntica a la función rgb, pero con un parámetro adicional denominado opacidad, para especificar la transparencia.
  • rgb(red, green, blue)
  • rgba(r,g,b,a)
 Ir a pagina
Modelo HSL
  • En este modelo utilizamos conceptos que son más fáciles de comprender.
  • Podemos definir la transparencia de un color usando la función hsla, esta función es idéntica a la función hsl, pero con un parámetro adicional denominado opacidad, para especificar la transparencia.
  • hsl(h,s,l)
  • hsla(h,s,l,a)

SELECTORES
Selectores Descripcíon Sintaxis
UNIVERSAL Selecciona todos los elementos. *{ declaraciones CSS }
TIPO Permite seleccionar elementos usando el nombre del elemento. Permite seleccionar elementos usando el nombre del elemento.
CLASE Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector. .valor_atributo { declaraciones CSS }
ID Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. #valor_atributo { declaraciones CSS }
ATRIBUTO Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. selector[attr] { … }
FLEXBOX
Propiedad Descripcíon
display La propiedad display con valor flex o inline-flex nos permite establecer un contenedor flexible. Los hijos del contenedor flexible se llaman elementos flexibles.
flex-direction La propiedad flex-direction permite establecer la dirección del eje principal del contenedor flexible. Los elementos flexibles se colocan a en la dirección del eje principal.
flex-wrap La propiedad flex-wrap controla si el contenedor flexible es de una sola línea o de varias líneas y la dirección del eje transversal, que determina la dirección en la que se alinean las líneas flexibles.
